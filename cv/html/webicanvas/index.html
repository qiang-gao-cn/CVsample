Webi UI overview

A webi UI can display in 3 modes. Open, Static Badge and Live Badge. Open mode and Static Badge mode are mandatory and all Webi badges need to support them. Live Badge mode is optional. The Open mode is simply a full Web GUI for that Webi. A Static Badge is like an icon that allows user to click to open a webi UI in Open, full page mode. Finally the optional Live Badge mode is similar to Live Tile of Windows, it occupy a small portion of the screen, more or less like the Static Badge except it shows some live data.

Badge sizes

Static badges comes in three sizes / layout. Small, Medium and Large size. Each badge occupy multiple number of 100 pixel X 100 pixel square grid. A Small badge occupy 1 square grid and is 100 pixel X 100 pixel size. A Large badge occupy 2X2 square grid and therefore is 200 pixel X 200 pixel size. There are two type of Medium badges . A Medium Horizontal badge occupy 2 X 1 grid with a 200 pixel width and 100 pixel height. A Medium Vertical badge occupy 1 X 2 grid with a 100 pixel width and 200 pixel height.

Badge Icons

You can place image icon inside each badge. All are shown in 70px square size, horizontally centered except large Badge. Large badge icon are in 120px square size. Underneath each badge icon contains a horizontally centered text to describe the badge.
